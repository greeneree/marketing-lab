# 🚀 마케팅랩 v2.1

소상공인을 위한 AI 맞춤 마케팅 전략 생성 서비스

**현재 배포 URL**: https://marketing-lab-ten.vercel.app/

---

## 📋 프로젝트 개요

### 목표
소상공인들이 쉽게 사용할 수 있는 **실전 마케팅 전략**을 AI가 자동으로 생성해주는 서비스

### 지원 업종
- 🏥 한의원
- ☕ 카페
- 💪 헬스장
- 💅 네일샵

### 주요 타겟
- 서울 마포구 소상공인
- 월 매출 500만원 ~ 5,000만원
- 마케팅 예산 10만원 ~ 300만원

## ✨ v2.1 주요 기능

### 1. 사전 질문 강화 (7개 필드 추가)
- ✅ **마포구 전체 동명 선택** (15개 법정동)
- ✅ **온라인 등록 여부** (네이버플레이스, 카카오맵, 기타, 없음)
- ✅ **히트 상품 유무** (있으면 이름 입력)
- 🆕 **주요 서비스 TOP 3 및 가격**
- 🆕 **경쟁 매장 3곳** (이름/특징/가격대)
- 🆕 **고객이 자주 묻는 질문** (최대 5개)
- 🆕 **예약/결제 방식** (네이버예약/전화/기타/현장)
- 🆕 **리뷰 수 및 평균 평점** (네이버/카카오/구글)
- 🆕 **가게의 강점 한 문장** (사장님 관점)

### 2. 12주 실행 계획 (캘린더)
- ✅ **90일 캘린더 형태** (3개월 일자별)
- ✅ **실천 체크 기능**: "실천했어요" 버튼 → "✅ 완료" 변경
- ✅ **프로그래스바**: 달성률 0% → 100% 실시간 표시
- ✅ **로컬스토리지 저장**: 브라우저 닫아도 기록 유지

### 3. AI 프롬프트 고도화 (v2.1)
- 🆕 **역할 강화**: "10년 이상 경력의 로컬 비즈니스 컨설턴트"
- 🆕 **근거 기반 전략**: 계산 가능한 수치는 계산식 제시
- 🆕 **가정 명시**: 모르는 정보는 (가정) 표시 + 검증 방법
- 🆕 **리스크 경고**: 업종별 법/플랫폼 정책 리스크 알림

### 4. 응답 구조 확장 (8개 섹션)
1. **핵심 결론 6줄 요약** (12주 핵심 레버 3개 + 하지 않을 것)
2. **현황 진단** (강점/약점 3단 구성 + 고객 세그먼트 + 포지셔닝)
3. **오퍼 설계 3개** (상품/패키지 + 전환/재방문 장치)
4. **즉시 실행 전략 7개** (KPI + 실행 체크리스트)
5. **동네상권 제휴 플랜 3개** (제휴 후보 + 파일럿 방법)
6. **12주 로드맵** (주차별 테마 + 일자별 액션 + 완료 기준)
7. **SNS/콘텐츠 12개** (포맷 + 훅 + 대본 + CTA)
8. **측정/대시보드** (KPI 8개 + 주간 회고 템플릿)

### 5. 마케팅 리소스 제공
- ✅ **인스타그램 해시태그** 추천 (15개)
- ✅ **네이버플레이스 키워드** 추천
- ✅ **카카오맵 키워드** 추천

### 6. 예상 성과 정확도 개선
- ✅ **숫자 오류 수정** (만원 → 원)
- ✅ **현재/3개월 후/증가분** 비교
- ✅ **ROI 범위 제시** (보수/기준/공격 시나리오)
- 🆕 **리스크 3개 + 완화책**

## 🚀 로컬 개발 환경 설정

### 1. 프로젝트 클론
```bash
git clone https://github.com/YOUR_USERNAME/marketing-lab.git
cd marketing-lab
```

### 2. 의존성 설치
```bash
npm install
```

### 3. 환경변수 설정
`.env` 파일 생성:
```
GOOGLE_API_KEY=AIza...
```

### 4. 로컬 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:3000` 접속
