## ⚠️ 알려진 문제

### React Router 경고
**현상**: 콘솔에 "No routes matched location" 경고 표시  
**원인**: React Router가 현재 URL 경로를 인식하지 못함  
**영향**: 빈 화면 표시 (앱이 렌더링되지 않음)  
**해결 방법**:
1. **Hash Router 사용** (권장): `BrowserRouter` → `HashRouter` 변경
2. **Base Path 설정**: Vite 설정에서 `base: '/'` 확인
3. **Catch-all 라우트 추가**: `<Route path="*" element={<MainPage />} />`

### 임시 해결책
현재는 정적 리소스 파일(MD, CSV)을 직접 다운로드하여 사용할 수 있습니다.

## 🎯 AI 연동 계획

### 정적 웹사이트 제약 사항
- ❌ **서버 사이드 처리 불가**: AI API 직접 호출 불가
- ❌ **환경 변수 숨기기 어려움**: API 키 노출 위험
- ❌ **CORS 문제**: 대부분 AI API는 브라우저 직접 호출 차단

### 가능한 해결책

#### Option 1: 클라이언트 측 프롬프트 생성기
사용자 입력 → JavaScript로 프롬프트 문자열 조합 → 텍스트 영역에 출력 → 사용자가 복사해서 ChatGPT에 붙여넣기
- ✅ 구현 간단
- ✅ AI API 키 불필요
- ❌ 사용자가 직접 ChatGPT에 가야 함

#### Option 2: 서버리스 함수 (Vercel/Netlify Functions)
사용자 입력 → 서버리스 함수 호출 → AI API 호출 → 결과 반환
- ✅ API 키 안전
- ✅ 사용자 경험 좋음
- ❌ 추가 인프라 필요
- ❌ 비용 발생 가능

#### Option 3: 외부 AI 챗봇 임베드
ChatGPT Custom GPT 또는 Claude Projects를 iframe으로 임베드
- ✅ 구현 간단
- ✅ AI 품질 보장
- ❌ 사용자가 ChatGPT Plus 계정 필요
