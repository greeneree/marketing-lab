# 업종별 맞춤 질문 설계

## 📋 전체 구조

```
Step 1: 기본 정보 (공통)
  ├─ 가게 이름
  ├─ 위치
  └─ 운영 기간

Step 2: 재무 정보 (공통)
  ├─ 월평균 매출
  ├─ 월평균 순이익
  └─ 평균 객단가

Step 3: 업종별 특화 질문 ⭐ NEW
  ├─ 한의원: 7개 질문
  ├─ 카페: 7개 질문
  ├─ 헬스장: 7개 질문
  └─ 네일샵: 7개 질문

Step 4: 고객 정보 (공통 - 일부 수정)
  ├─ 주요 고객 성별/연령
  ├─ 재방문율
  └─ 유입 경로

Step 5: 운영 현황 (공통)
  ├─ 직원 수
  ├─ 바쁜 시간대
  └─ SNS 활동

Step 6: 고민과 목표 (공통)
  ├─ 현재 고민
  ├─ 3개월 목표
  └─ 마케팅 예산
```

## 🏥 한의원 전용 질문 (Step 3)

### 질문 1: 건강보험 적용 비율
**질문**: "건강보험이 적용되는 진료 비율은 어느 정도인가요?"

**선택지**:
- [ ] 10% 미만 (대부분 비급여)
- [ ] 10-30%
- [ ] 30-50%
- [ ] 50% 이상

**활용**: 가격 민감도, 타겟 고객층 파악

---

### 질문 2: 주력 진료 분야
**질문**: "어떤 진료에 강점이 있나요? (복수선택 가능)"

**선택지**:
- [ ] 통증 치료 (허리, 목, 어깨 등)
- [ ] 여성 질환 (산후조리, 생리통 등)
- [ ] 다이어트/체형 관리
- [ ] 안면 침술/미용 한의학
- [ ] 만성 질환 관리 (소화불량, 불면 등)
- [ ] 특별한 강점 없음

**활용**: 콘텐츠 방향, 타겟 키워드 선정

---

### 질문 3: 초진 vs 재진 비율
**질문**: "내원 환자 중 초진과 재진의 비율은?"

**입력 방식**: 슬라이더 (초진 0% ←→ 100% 재진)

**활용**: 
- 초진 많음 → 신규 유입은 되지만 단골 안 됨
- 재진 많음 → 단골은 있지만 신규 유입 부족

---

### 질문 4: 평균 치료 횟수
**질문**: "환자들이 평균 몇 회 정도 치료를 받나요?"

**선택지**:
- [ ] 1-3회 (단기 치료 위주)
- [ ] 4-10회 (중기 치료)
- [ ] 10회 이상 (장기 치료 계획)
- [ ] 잘 모르겠음

**활용**: LTV(고객 생애 가치) 계산, 패키지 상품 제안

---

### 질문 5: 차별화 포인트
**질문**: "주변 한의원과 비교했을 때 우리의 강점은?"

**선택지**:
- [ ] 특정 질환에 전문성 (예: 디스크 전문)
- [ ] 최신 장비 보유 (체외충격파, 도수치료 등)
- [ ] 가격이 저렴함
- [ ] 원장님 경력/인지도
- [ ] 친절하고 편안한 분위기
- [ ] 잘 모르겠음

**활용**: USP(독특한 판매 제안) 도출

---

### 질문 6: 주변 환경
**질문**: "한의원 주변 환경은 어떤가요?"

**선택지**:
- [ ] 병원가 (의료기관 밀집)
- [ ] 주거 지역 (아파트 단지)
- [ ] 오피스 밀집 지역 (직장인 많음)
- [ ] 상권 지역 (유동인구 많음)

**활용**: 시간대별 전략, 타겟 세분화

---

### 질문 7: 온라인 예약 시스템
**질문**: "온라인 예약이 가능한가요?"

**선택지**:
- [ ] 네, 환자들이 많이 사용함
- [ ] 있지만 거의 사용 안 됨 (전화 선호)
- [ ] 없음 (전화 예약만 가능)

**활용**: 디지털 전환 수준 파악, 개선 포인트

## ☕ 카페 전용 질문 (Step 3)

### 질문 1: 카페 컨셉
**질문**: "어떤 스타일의 카페인가요?"

**선택지**:
- [ ] 디저트 맛집 (예쁜 케이크/디저트 중심)
- [ ] 조용한 스터디 카페 (공부/작업하기 좋음)
- [ ] 브런치 카페 (식사 메뉴 강점)
- [ ] 반려동물 동반 가능
- [ ] 특별한 컨셉 없는 동네 카페
- [ ] 기타: [직접 입력]

**활용**: 타겟 고객, SNS 콘텐츠 방향

---

### 질문 2: 시그니처 메뉴
**질문**: "시그니처 메뉴가 있나요?"

**선택지**:
- [ ] 네, 이것 때문에 찾아오는 손님 많음
- [ ] 있지만 잘 안 알려짐
- [ ] 아직 개발 중
- [ ] 없음

**추가 입력** (있다면): [시그니처 메뉴 이름]

**활용**: 마케팅 포커스 포인트

---

### 질문 3: 매출 구성
**질문**: "매출 구성 비율은 어떻게 되나요?"

**입력 방식**: 3개 슬라이더 (합계 100%)
- 커피/음료: [  ]%
- 디저트: [  ]%
- 브런치/식사: [  ]%

**활용**: 개선 여지가 있는 카테고리 파악

---

### 질문 4: 평일 vs 주말
**질문**: "평일과 주말 매출 차이가 얼마나 나나요?"

**선택지**:
- [ ] 주말이 압도적 (평일 한산함)
- [ ] 주말이 약간 높음
- [ ] 비슷함
- [ ] 평일이 더 높음 (직장인 고객)

**활용**: 평일 매출 올리기 전략

---

### 질문 5: 인스타그램 노출
**질問**: "인스타그램에서 우리 카페를 검색하면?"

**선택지**:
- [ ] 손님들이 올린 사진 많음 (월 10장 이상)
- [ ] 가끔 보임 (월 1-5장)
- [ ] 거의 없음
- [ ] 확인 안 해봄

**활용**: 인스타그래머블 요소 개선 필요성

---

### 질문 6: 테이크아웃 비율
**질문**: "테이크아웃과 매장 이용 비율은?"

**입력 방식**: 슬라이더 (테이크아웃 0% ←→ 100% 매장)

**활용**: 공간 활용 전략, 회전율 개선

---

### 질문 7: 경쟁 강도
**질문**: "주변 카페 경쟁은 얼마나 심한가요?"

**선택지**:
- [ ] 매우 심함 (반경 100m 내 5개 이상)
- [ ] 보통 (반경 200m 내 2-4개)
- [ ] 약함 (주변에 거의 없음)

**활용**: 차별화 전략의 강도 결정

---

## 💪 헬스장 전용 질문 (Step 3)

### 질문 1: 운영 형태
**질문**: "어떤 형태로 운영하시나요?"

**선택지**:
- [ ] PT 전문 (회원권 판매 안 함)
- [ ] 회원권 + PT 병행
- [ ] 회원권 위주 (PT는 소수만)

**활용**: 비즈니스 모델에 따른 전략 차별화

---

### 질문 2: PT 비율
**질문**: "전체 회원 중 PT를 받는 비율은?"

**입력 방식**: 슬라이더 (PT 0% ←→ 100% 일반회원)

**활용**: 수익성 개선 포인트

---

### 질문 3: 회원권 구성
**질문**: "회원권은 주로 어떤 기간으로 등록하나요?"

**선택지**:
- [ ] 1개월 단위가 대부분
- [ ] 3-6개월 중기 회원 많음
- [ ] 12개월 이상 장기 회원 있음

**활용**: 장기 회원 유도 전략

---

### 질문 4: 등록 회원 수
**질문**: "현재 등록된 회원은 몇 명인가요?"

**선택지**:
- [ ] 50명 미만
- [ ] 50-100명
- [ ] 100-200명
- [ ] 200명 이상

**활용**: 규모에 따른 전략 (소규모 vs 대형)

---

### 질문 5: 재등록률
**질문**: "회원권 만료 후 재등록하는 비율은?"

**선택지**:
- [ ] 30% 미만 (대부분 끊김)
- [ ] 30-50%
- [ ] 50-70%
- [ ] 70% 이상

**활용**: 가장 중요한 지표! 재등록 전략 필수

---

### 질문 6: 시설 특징
**질문**: "우리 헬스장의 강점은? (복수선택)"

**선택지**:
- [ ] GX룸/그룹 수업 운영
- [ ] 깨끗한 샤워실/락커룸
- [ ] 최신 운동 기구
- [ ] 24시간 운영
- [ ] 여성 전용 공간/시간
- [ ] 무료 주차 가능
- [ ] 특별한 강점 없음

**활용**: 차별화 포인트 강조

---

### 질문 7: 신규 회원 유입
**질문**: "신규 회원은 주로 어떻게 들어오나요?"

**선택지**:
- [ ] 기존 회원 추천이 대부분
- [ ] 네이버/카카오맵 검색
- [ ] 지나가다 발견
- [ ] 전단지/현수막
- [ ] 인스타그램
- [ ] 잘 모르겠음

**활용**: 효과적인 채널에 집중

## 💅 네일샵 전용 질문 (Step 3)

### 질문 1: 주력 서비스
**질문**: "어떤 서비스가 주력인가요?"

**선택지**:
- [ ] 기본 젤 네일 (심플 디자인)
- [ ] 아트 네일 (복잡한 디자인 특화)
- [ ] 속눈썹/왁싱 병행
- [ ] 페디큐어 비중 높음
- [ ] 네일 케어/관리

**활용**: 타겟 고객 세분화

---

### 질문 2: 평균 시술 시간
**질문**: "손님 한 분 시술에 평균 얼마나 걸리나요?"

**선택지**:
- [ ] 1시간 미만 (빠른 회전)
- [ ] 1-1.5시간
- [ ] 1.5-2시간
- [ ] 2시간 이상 (정교한 아트)

**활용**: 예약 시스템 최적화

---

### 질문 3: 재시술 주기
**질문**: "단골 손님들은 보통 몇 주마다 오시나요?"

**입력 방식**: 숫자 입력 [  ]주

**활용**: 가장 중요한 수익 지표!

---

### 질문 4: 예약 방식
**질문**: "예약은 어떻게 받으시나요?"

**선택지**:
- [ ] 100% 예약제만 운영
- [ ] 예약 우선, 당일 가능하면 받음
- [ ] 워크인(당일 방문) 손님도 많음

**활용**: 운영 효율화 전략

---

### 질문 5: 가격대
**질문**: "우리 샵의 가격 포지셔닝은?"

**선택지**:
- [ ] 저가 (기본 3만원대)
- [ ] 중가 (기본 4-5만원대)
- [ ] 고가 (기본 6만원 이상)

**활용**: 타겟 고객층, 프로모션 전략

---

### 질문 6: 디자이너 수
**질문**: "디자이너는 몇 분이신가요?"

**선택지**:
- [ ] 1인 (원장님만)
- [ ] 2명
- [ ] 3명 이상

**활용**: 확장 가능성, 예약 관리 전략

---

### 질문 7: 포트폴리오 관리
**질문**: "인스타그램에 시술 사진을 올리시나요?"

**선택지**:
- [ ] 네, 거의 매일 올림 (주 5회 이상)
- [ ] 주 2-3회 올림
- [ ] 가끔 올림 (월 2-3회)
- [ ] 거의 안 올림
- [ ] 인스타그램 안 함

**활용**: 네일샵은 비주얼이 생명!
