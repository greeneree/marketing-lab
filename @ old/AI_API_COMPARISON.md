## 💰 실제 비용 시뮬레이션

### 요청당 평균 토큰 사용량 추정
```
사용자 입력: ~500 토큰
프롬프트 템플릿: ~2,000 토큰
AI 응답 (전략 5개 + 로드맵): ~3,000 토큰
--------------------------------------------
총 토큰: ~5,500 토큰 (약 5.5K)
```

### OpenAI (GPT-4o) 비용
```
입력: 2,500 토큰 × $0.0025/1K = $0.00625
출력: 3,000 토큰 × $0.01/1K = $0.03
--------------------------------------------
요청당: $0.036 (약 48원)

월 100명: $3.6 (4,800원)
월 500명: $18 (24,000원)
월 1,000명: $36 (48,000원)
```

### Anthropic (Claude 3.5 Sonnet) 비용
```
입력: 2,500 토큰 × $0.003/1K = $0.0075
출력: 3,000 토큰 × $0.015/1K = $0.045
--------------------------------------------
요청당: $0.053 (약 70원)

월 100명: $5.3 (7,000원)
월 500명: $26.5 (35,000원)
월 1,000명: $53 (70,000원)
```

### Google (Gemini 1.5 Pro) 비용
```
입력: 2,500 토큰 × $0.00125/1K = $0.003
출력: 3,000 토큰 × $0.005/1K = $0.015
--------------------------------------------
요청당: $0.018 (약 24원)

월 100명: $1.8 (2,400원)
월 500명: $9 (12,000원)
월 1,000명: $18 (24,000원)
```

### Groq (Llama 3.1 70B) 비용
```
무료 (단, 제한 있음)
- 일일 요청 제한: ~14,400 요청
- 분당 요청 제한: 30 요청
- 안정성: 보통 (가끔 다운타임)
--------------------------------------------
요청당: $0 (무료!)

월 100명: 무료
월 500명: 무료 (단, 안정성 리스크)
월 1,000명: 무료 (단, 속도 제한 걸릴 수 있음)
```
