## 🎨 주요 기능 상세

### 1. 사전 질문 폼
```html
✅ 업종 선택 (한의원, 카페, 헬스장, 네일샵)
✅ 가게 이름
✅ 위치 (마포구 15개 동)
✅ 월 평균 매출
✅ 온라인 등록 여부 (체크박스 4개)
✅ 히트 상품 유무 (라디오 버튼 + 조건부 입력필드)
✅ 현재 고민사항
```

### 2. 12주 실행 계획 캘린더
```javascript
// 90일 캘린더 (3개월)
- 일자별 실행 과제 표시
- 요일 표시 (월~일)
- 실천했어요 버튼
- 완료 시 배경색 변경 (흰색 → 초록색)
- 로컬스토리지 저장 (브라우저 닫아도 유지)
```

### 3. 프로그래스바
```css
달성률 표시: 0% → 100%
- 총 과제 수 계산
- 완료된 과제 수 카운트
- 실시간 퍼센트 업데이트
- 그라데이션 진행바
```

### 4. AI 프롬프트 강화
```
기존: "인스타그램 게시물 올리기"
개선: "망원동 카페 3곳과 제휴하여 상호 할인 쿠폰 제공"

기존: "SNS 마케팅 강화"
개선: "망원라떼 시그니처 메뉴로 브랜딩 + 포토존 제작"

기존: "고객 관리 시스템"
개선: "헬스장-한의원 제휴로 통증 치료 고객 20% 할인"
```

## 🧪 테스트 시나리오

### 시나리오 1: 카페 (히트상품 있음)
```
업종: 카페
이름: 망원카페
위치: 망원동
월 매출: 300만원
온라인: 네이버플레이스, 카카오맵
히트상품: 있다 → "망원라떼"
고민: 평일 손님이 없어요
```

**예상 AI 응답**:
- 전략 1: 망원라떼 시그니처 메뉴 강화 (포토존 제작)
- 전략 2: 평일 오전 재택근무족 타겟 할인
- 전략 3: 망원동 상권 제휴 (헬스장, 미용실)
- 해시태그: #망원카페 #망원라떼 #망원동카페
- 키워드: "망원동 카페", "망원 시그니처"

### 시나리오 2: 헬스장 (히트상품 없음)
```
업종: 헬스장
이름: 파워짐
위치: 합정동
월 매출: 2000만원
온라인: 네이버플레이스
히트상품: 없다
고민: PT 등록률이 낮아요
```

**예상 AI 응답**:
- 전략 1: 신규 시그니처 프로그램 개발 (100일 챌린지)
- 전략 2: 인근 한의원 제휴 (재활 운동 프로그램)
- 전략 3: PT 무료 체험 1회 제공
- 해시태그: #합정헬스장 #합정PT #파워짐
- 키워드: "합정 헬스장", "합정역 PT"
