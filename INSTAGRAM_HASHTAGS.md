# 📸 인스타그램 해시태그 추천 키워드 (업종별 30개)

## 🎯 사용 목적
인스타그램 포스트에 복사해서 붙여넣기만 하면 되는 **업종별 맞춤 해시태그**입니다.

---

## 📋 해시태그 사용 가이드

### ✅ 기본 원칙
1. **개수:** 포스트당 20~30개 (인스타 최대 30개)
2. **조합:** 대형 태그 5개 + 중형 태그 10개 + 소형 태그 15개
3. **위치:** 본문 끝 또는 첫 댓글에 입력
4. **언어:** 한글 태그 위주 (한국 타겟)

### 📊 태그 크기 분류
- **대형 태그:** 게시물 100만 개 이상 (예: #카페, #맛집)
- **중형 태그:** 게시물 10만~100만 개 (예: #홍대카페, #마포맛집)
- **소형 태그:** 게시물 1만~10만 개 (예: #망원동카페, #공덕카페)

## 💡 해시태그 전략

### 1. 조합 공식
```
기본 해시태그 (10개) +
지역 해시태그 (5개) +
타겟/컨셉 해시태그 (10개) +
트렌드 해시태그 (5개)
= 총 30개
```

### 2. 주기적 교체
- **매주 5개씩** 트렌드 태그 교체
- 계절별 태그 업데이트 (봄/여름/가을/겨울)
- 이벤트 태그 추가 (명절, 기념일)

### 3. 금지 태그 (인스타 규정 위반)
❌ `#좋아요` `#맞팔` `#선팔` `#소통` `#인친`
→ 이런 태그는 노출 감소 원인!

### 4. 효과 측정
- 일주일 후 인사이트 확인
- 도달 수, 저장 수 높은 태그 유지
- 효과 없는 태그 교체

## ❓ 자주 묻는 질문

### Q1: 해시태그는 본문에 넣어야 하나요, 댓글에 넣어야 하나요?
**A:** 둘 다 효과는 같습니다. 본문이 깔끔하게 보이려면 첫 댓글에 입력하세요.

### Q2: 30개 다 써야 하나요?
**A:** 아니요. 최소 10개 이상 권장. 30개가 부담스러우면 20개로 시작하세요.

### Q3: 같은 해시태그를 계속 써도 되나요?
**A:** 네, 괜찮습니다. 하지만 매주 5개 정도는 교체하는 게 좋습니다.

### Q4: 영어 해시태그도 넣어야 하나요?
**A:** 한국 손님만 타겟이면 한글만으로 충분합니다.

### Q5: 해시태그 순서가 중요한가요?
**A:** 아니요. 순서는 상관없습니다.
